# NovoeTV Player UI (demo 4.000)

Интерактивное превью плеера NovoeTV APK 4.000 для Samsung TV.

**Демо:** https://ryt1nger.github.io/novoetv-player-ui-demo/ui/1920/index.html

## Структура

| Путь | Назначение |
|------|------------|
| `ui/1920/index.html` | Точка входа |
| `ui/js/` | PlayerUI, EpgUI, SepgUI, AppPreview |
| `ui/1920/css/` | Стили v4 + `mvideo.ttf` |
| `assets/v4/` | Иконки и изображения APK |
| `assets/img/` | Фон видео и спиннер |
| `mocks/` | Mock-данные и постеры |

Внизу справа — компактный **пульт (Remote)** для навигации; debug-состояния скрыты под кнопкой Debug. Для скриншотов без пульта: `?audit&state=live`.
